# ControleF — Backend (NestJS)

API REST do sistema de controle financeiro pessoal.

## Stack
- **NestJS** com arquitetura modular
- **Prisma 5** + **PostgreSQL**
- **Swagger** disponível em `/api`
- Pronto para integração com **n8n**

## Setup

### 1. Pré-requisitos
- Node.js 18+
- PostgreSQL rodando localmente (ou via Docker)

### 2. Variáveis de ambiente
```bash
cp .env.example .env
# Edite DATABASE_URL com suas credenciais
```

### 3. Banco de dados
```bash
# Criar e aplicar migrations
npx prisma migrate dev --name init

# Ou apenas sincronizar o schema (sem migrations)
npx prisma db push
```

### 4. Rodar em desenvolvimento
```bash
npm run start:dev
```

### 5. Rodar em produção
```bash
npm run build
npm run start:prod
```

## Módulos e Endpoints

| Módulo | Prefixo | Descrição |
|---|---|---|
| accounts | `/accounts` | Contas recorrentes (água, energia...) |
| installments | `/installments` | Parcelamentos com geração automática de parcelas |
| expenses | `/expenses` | Gastos esporádicos |
| investments | `/investments` | Investimentos com resumo consolidado |
| monthly-control | `/monthly-control` | Controle mensal + dashboard |

### Endpoints n8n-ready
- `POST /expenses` — Criar gasto via webhook (Telegram/WhatsApp)
- `GET /monthly-control/dashboard` — Resumo financeiro do mês atual

## Swagger
Acesse `http://localhost:3000/api` para a documentação interativa.

## Docker (PostgreSQL)
```bash
docker run --name controlef-db -e POSTGRES_PASSWORD=postgres -e POSTGRES_DB=controlef -p 5432:5432 -d postgres
```
